# Factory Method

## Objetivo

Implementar um sistema de notificações usando o padrão criacional Factory Method.

O código cliente utiliza uma fábrica para criar notificações, evitando instanciar diretamente as classes concretas.

## Saída esperada

```txt
EmailFactory enviando: "Bem-vindo ao sistema!"
SMSFactory enviando: "Bem-vindo ao sistema!"
PushFactory enviando: "Bem-vindo ao sistema!"
```
